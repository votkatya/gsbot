# Инструкция для администраторов — «Город Спорта»

---

## Что такое этот сервис

«Город Спорта» — это мини-приложение для клиентов фитнес-клуба, доступное через Telegram и ВКонтакте. Суть: клиент выполняет задания, связанные с посещением клуба, и получает за это виртуальные спорткоины. На спорткоины можно купить призы.

**Цель** — вовлечь новых клиентов в жизнь клуба, познакомить со всеми зонами и услугами.

---

## Как клиент взаимодействует с приложением

### 1. Регистрация
Клиент открывает приложение через Telegram или ВКонтакте, вводит имя и номер телефона. Без регистрации задания недоступны.

### 2. Задания
Задания выполняются последовательно, блок за блоком — одно за другим по порядку. Каждое задание связано с зоной или услугой клуба:
- Посетить тренажёрный зал
- Пройти Таниту
- Полежать в джакузи
- Посетить групповое занятие
- Оставить отзыв на Яндекс.Картах
- и другие

Чтобы подтвердить выполнение, клиент вводит код или загружает скриншот.

### 3. Как клиент подтверждает задание

| Тип задания | Что делает клиент |
|---|---|
| **QR / Ручной код** | Сканирует QR-код на стойке в зоне или вводит короткий код вручную (например, `TNT45`) |
| **Код приложения** | Находит код в мобильном приложении клуба и вводит его |
| **Фото** | Делает скриншот отзыва на Яндекс.Картах и загружает в приложение |
| **Квиз** | Отвечает на вопросы о клубе |

### 4. Монеты и магазин
После выполнения задания клиент получает спорткоины (🪙). Накопив достаточно, переходит в магазин, добавляет приз в корзину и оплачивает спорткоинами. После оплаты — показывает код `GS-XXXXXX` сотруднику на стойке.

## Панель администратора

**Адрес:** `https://gsbot18.ru/admin/`

Войти можно только по паролю. Есть два уровня доступа:
- **admin** — полный доступ: одобрять/отклонять, редактировать, выдавать призы
- **viewer** — только просмотр

### Дашборд

Главная страница. Показывает общую статистику:
- Сколько зарегистрировалось пользователей
- Сколько заданий выполнено
- Сколько покупок совершено
- График активности за последнюю неделю

Заходить сюда не обязательно — это просто обзорная информация.

### Проверка отзывов ⚡ Самое важное для ежедневной работы

Сюда попадают скриншоты от клиентов, которые выполнили задание «Оставь отзыв на Яндекс.Картах».

**Что делать:**

1. Открыть вкладку **«Отзывы»**
2. По умолчанию показываются заявки **«На проверке»** — именно их нужно обработать
3. Для каждой заявки: посмотреть скриншот (нажать для увеличения), убедиться что отзыв настоящий
4. Нажать **✅ Одобрить** — клиенту начисляются спорткоины, он получает уведомление
5. Или нажать **❌ Отклонить** — ввести причину (она придёт клиенту), например: «Скриншот нечитаем» или «Отзыв не найден на картах»

> **Важно:** список обновляется автоматически каждые 30 секунд. Если заявок много — видна оранжевая плашка с количеством.

**Фильтры:** можно переключаться между «На проверке», «Одобренные», «Отклонённые», «Все».

### Покупки — выдача призов

Сюда попадают все покупки клиентов из магазина.

**Когда клиент подходит на стойку с кодом `GS-XXXXXX`:**

1. Открыть вкладку **«Покупки»**
2. Найти в таблице **«Ожидают выдачи»** нужный код
3. Убедиться что имя пользователя совпадает
4. Нажать **Выдать ✓** — покупка переходит в раздел «Выданные»

> Список обновляется каждые 15 секунд автоматически.

**Выданные** — история всех выданных призов (для справки).

### Задания

Список всех заданий программы. Здесь можно:

- Посмотреть тип каждого задания, его описание и сколько человек выполнило
- Нажать **✏️ Изменить** для редактирования задания

**Что можно редактировать:**
- Название и описание задания
- Размер награды в спорткоинах
- Тестовый код (для проверки работы — не нужен клиентам)
- Основной код (QR и ручной) — один код используется и для QR-стикера, и для ручного ввода

> **Тестовый код** (`TEST1`) — используйте его сами, чтобы проверить что задание работает. Клиентам его не давайте.

## Типичный рабочий день администратора

1. **Утром** — зайти в «Отзывы», проверить новые скриншоты, одобрить или отклонить
2. **В течение дня** — если клиент подходит на стойку с кодом GS-XXXXXX, зайти в «Покупки» и нажать «Выдать»
3. **По необходимости** — редактировать задания (если меняются коды или условия)

## Частые ситуации

**Клиент говорит «я выполнил задание, но спорткоины не пришли»**
→ Зайти в Пользователи → найти клиента → посмотреть его задания. Если задание не отмечено — можно отметить вручную кнопкой «Выполнено».

**Клиент говорит «мой скриншот отклонили, но отзыв настоящий»**
→ Зайти в Отзывы → вкладка «Отклонённые» → найти заявку → если отзыв действительно настоящий, сообщить разработчику (через админку нет кнопки «переотправить»).

**Нужно изменить QR-код в зоне клуба**
→ Задания → найти нужное задание → Изменить → обновить код → распечатать новый QR-стикер.

**Клиент не может войти в приложение**
→ Убедиться что он открывает через Telegram или ВКонтакте, а не обычный браузер. Приложение работает только как мини-приложение внутри этих платформ.
